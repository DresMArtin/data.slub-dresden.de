# SLUB LOD API documentation
The API provided by [data.slub-dresden.de](http://data.slub-dresden.de) for Linked Open Data can be used to process datasets from different sources (K10Plus, SWB), enriched with norm data from multiple authority providers (GND, wikidata, SWB, â€¦).

There is an [interactive UI](http://data.slub-dresden.de/api) where you can experience the API directly and try out some request with various options before coding them.
